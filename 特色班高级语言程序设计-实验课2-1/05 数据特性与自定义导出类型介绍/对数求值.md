#验证数据的可变性对程序的影响

从控制台输入 $n(-32\leq n\leq 32)$，然后将 $n^{3}$ 赋值给 $n$，再将 $n^{2}$ 赋值给 $n$ 
在每次操作后，都输出 `log(n)` 的值，如果对应时刻的 $n$ 不符合对数运算的要求，请输出 `error`

#### 样例输入 1
```
5
```
#### 样例输出 1
```
1.60944
4.82831
9.65663
```
#### 样例输入 2
```
-5
```
#### 样例输出 2
```
error
error
9.65663
```
